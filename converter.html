How to Use the Distance Converter

  1. Enter the distance in the input field. You can input the value in either miles or kilometers.

  2. Select the desired conversion direction using the radio buttons: S Miles to Kilometers: Converts miles to kilometers. Kilometers to Miles: Converts kilometers to miles.

Click the "Convert" button, and the result will be displayed below the button.
